# 🎮 InternPe Task – Rock Paper Scissors Web Game

This project is a fully functional **Rock Paper Scissors game** developed as part of the **InternPe Internship Task** using core web technologies.

## 🚀 Features
- User vs Computer gameplay  
- Interactive Rock ✊ Paper ✋ Scissors ✌️ selection  
- Turn-based system with real-time feedback  
- Dynamic score tracking for both user and computer  
- Result display (Win / Lose / Draw) after each round  
- Clean circular UI inspired by classic hand-gesture games  
- Smooth transitions for a real-game experience  
- Fully center-aligned responsive layout  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## 📌 Learning Outcomes
- DOM manipulation using JavaScript  
- Implementing conditional game logic  
- Handling user interaction and events  
- Managing real-time UI updates  
- Improving UI/UX for interactive web applications
