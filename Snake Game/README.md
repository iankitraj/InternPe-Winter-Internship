# 🐍 InternPe Task – Modern Snake Game

This project is a fully functional **Snake Game** developed as part of the **InternPe Internship Task** using core web technologies.  
The game recreates the classic Snake gameplay with **modern UI improvements, animations, and dynamic difficulty levels**.

## 🚀 Features
- Classic Snake gameplay using keyboard controls  
- **Animated food** with pulsing effect  
- **Snake with rounded body and eyes** for modern visuals  
- **Dynamic speed increase every 5 points**  
- **Level system** that increases game difficulty  
- **High score tracking using LocalStorage**  
- **Game Over screen with restart option**  
- **Start game button for controlled gameplay**  
- **Background grid for better visual gameplay experience**  
- Clean and responsive game layout  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- HTML5 Canvas API  

## 📌 Learning Outcomes
- Implementing game logic using JavaScript  
- Working with **HTML5 Canvas for rendering game elements**  
- Managing **keyboard events for player movement**  
- Implementing **collision detection algorithms**  
- Using **LocalStorage for persistent high score tracking**  
- Building **interactive animations using JavaScript**  
- Improving **UI/UX for browser-based games**

## 🎮 How to Play
1. Click **Start Game**  
2. Use **Arrow Keys** to control the snake  
3. Eat the food to increase your score  
4. Every **5 points increases the game speed and level**  
5. Avoid hitting the walls or the snake's own body
