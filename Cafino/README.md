# ☕ InternPe Task – CoffeeBliss Coffee Shop Website

This project is a modern **Coffee Shop Landing Page** developed as part of the **InternPe Internship Task (Week 11 & 12)** using core web technologies.  
The website showcases a stylish coffee shop interface with **interactive UI components, smooth animations, popup modals, and responsive design**.

The goal of this project was to practice **frontend development concepts, DOM manipulation, and modern UI/UX design** using pure HTML, CSS, and JavaScript.

---

## 🚀 Features
- Modern **coffee shop landing page design**
- Sticky **navigation bar with smooth scrolling**
- Animated **hero section background slider**
- Interactive **coffee menu cards**
- **Order popup modal** for placing orders
- **Login / Sign Up modal UI**
- Coffee **image gallery with hover effects**
- **Contact form with basic validation**
- Order confirmation message system
- **Dynamic footer year using JavaScript**
- Clean and responsive UI layout

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- Google Fonts (Poppins)

---

## 📌 Learning Outcomes
- Building a complete **landing page UI from scratch**
- Implementing **modal popups using JavaScript**
- Handling **form submission and validation**
- Using **DOM manipulation and event listeners**
- Creating **responsive layouts with Flexbox and Grid**
- Implementing **CSS animations and hover effects**
- Structuring and organizing frontend project files

---

## 🌐 Website Sections
1. **Home (Hero Section)** – Animated coffee background slider  
2. **Menu** – Coffee cards with order buttons  
3. **Gallery** – Coffee images with hover effects  
4. **About** – Information about the coffee shop  
5. **Contact** – Contact form for user messages
